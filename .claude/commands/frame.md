Read `framing-doc/SKILL.md` first and follow it as the primary instruction for this command.

Use this slash command when the user wants to turn messy source material into a durable frame before shaping or implementation.

User request and source context:

$ARGUMENTS

Produce or update a framing artifact that keeps source evidence separate from interpretation and includes, as applicable:

- Source
- Pre-work / option landscape
- Problem
- Outcome
- Boundaries
- Less about / More about
- Self-check

Do not move into shaping, breadboarding, or implementation unless the user explicitly asks after the frame is complete.
