Read `kickoff-doc/SKILL.md` first and follow it as the primary instruction for this command.

Use this slash command when the team has already converged enough that builders need a durable reference doc.

User request and source context:

$ARGUMENTS

Produce or update a kickoff artifact that includes:

- Frame
- Shaped territory sections
- Decisions attached to the system area they affect
- Constraints and exclusions
- Open questions that still matter
- Context Card when it will feed downstream agent work

Organize by the territory being built, not by meeting order or build sequence.
