# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-05-01

Initial public release of Planning Skills for Agents and Humans.

### Included

- Six reusable planning skills:
  - `framing-doc`
  - `shaping`
  - `breadboarding`
  - `breadboard-reflection`
  - `kickoff-doc`
  - `feed-planning-context`
- Tool-neutral agent instructions in `AGENTS.md`.
- Claude-oriented instructions in `CLAUDE.md`.
- Claude plugin manifest in `.claude-plugin/plugin.json`.
- Codex plugin manifest in `.codex-plugin/plugin.json`.
- Repo-local Codex marketplace entry in `.agents/plugins/marketplace.json`.
- Gemini/MCP integration notes under `integrations/gemini/`.
- Optional TypeScript MCP server under `mcp-server/`.
- Canvas export contract for Mermaid, FigJam, Excalidraw, TLDraw, and Miro-style visual surfaces.
- Example planning artifacts for a simple grocery-list feature.
